# RPN Calculator

## What is This

This is a tool to perform calculations on a terminal using Reverse Polish Notation.
The values are kept on the stack. The values are kept on the stack and are fetched from the end of the stack.

## Usage example
To do 2 + 3

    2 3 + Enter

or

    2 Enter
    3 Enter
    + Enter

type.
